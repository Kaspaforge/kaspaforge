Сейф, в котором кражу можно отменить

Кастодия — по буквам

Шифруется локально.
Синхронизируется только шифротекст.

Общие данные Деска живут в age-зашифрованном профиле под твоим паролем. Forge Sync только после твоего включения отправляет на сервер только шифротекст — сервер не может его расшифровать. Alarm-ключи с раздельным хранением в зеркало не попадают.

  • Ключи остаются в браузере
    Создание, расшифровка и подпись происходят локально. Пароль и открытые ключевые данные не отправляются Forge.
  • Один экспорт восстанавливает общий профиль
    Зашифрованный экспорт .age восстанавливает общие данные Деска. Alarm-карточки остаются отдельно и требуют собственной защищённой копии.
  • Мы никогда не просим ключи
    Ни пароль, ни приватные ключи, ни файл ключей — ни в чате, ни по почте, ни «для проверки». Кто просит — мошенник.
  • Открытый код, проверяемые слова
    Контракты и инструменты опубликованы: github.com/Kaspaforge/kaspaforge. Не верь тексту на этой странице — проверь.
Построено, чтобы пережить нас
Правила форсит сама сеть Kaspa, а не наши серверы. Если сайт завтра исчезнет: сейф продолжит работать через открытый CLI vaultctl с любой нодой Kaspa v2+, а каждый эскроу развяжется сам — таймауты вернут средства стороне по умолчанию. Потеря нас стоит тебе сайта, а не денег.
Деск

Один деск управляет всем

Деск — твоя рубка управления: личный Kaspa-кошелёк (приём, баланс, отправка), из которого напрямую финансируются сейфы и сделки, портфель сейфов с живым статусом, твои сделки и объявления — всё расшифровывается из одного профиля, всё в одном месте.

Монеты ходят по кругу: храни в Сейфе, закладывай сделку в Эскроу, продавай на Маркете — выручка возвращается на деск, а дальше решаешь ты.
